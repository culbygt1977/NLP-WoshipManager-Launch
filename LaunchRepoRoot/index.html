Install / Update NLP Worship Manager
Windows Security message?
If you see “Windows protected your PC”, click More info → Run anyway.

Install steps

  1. Click the blue Install / Update button above.
  2. If your browser downloads a file, open the downloaded .application file to start the install.
  3. Follow the prompts to complete installation.
  4. After installing, launch the app from the Start Menu.

Updates

Once installed, updates are automatic. Just open the app from the Start Menu and it will check for updates before starting.

Troubleshooting

  1. Nothing happens when clicking Install: download the file and open the .application from Downloads.
  2. Install fails or keeps using an old version: uninstall the app, then reinstall from this page.
  3. Still stuck? Contact Greg for help.
